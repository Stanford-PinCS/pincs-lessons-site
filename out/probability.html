What is Probability?

Probability measures how likely something is to happen. It's always a number between 0 and 1:

  • 0 (0%) = Impossible event
  • 0.5 (50%) = Equal chance of happening or not happening
  • 1 (100%) = Certain event

We can express probability as a fraction, decimal, or percentage.

The basic formula: Probability = Number of favorable outcomes / Total number of possible outcomes

Key Probability Concepts

Equal vs. Unequal Probability

Some events (like coin flips) have equal probabilities for all outcomes. Others (like M&M colors) have different probabilities for different outcomes.

Independent Events

Each coin flip or dice roll is independent - previous results don't affect future probabilities. The coin has no "memory" of past flips.

Law of Large Numbers

As you increase the number of trials, the actual results will get closer to the theoretical probability.

Representing Probability

We can express the same probability as a fraction (1/2), decimal (0.5), or percentage (50%).
