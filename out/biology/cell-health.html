Function in Computer Science:

A function is a reusable block of code that performs a specific task. It typically takes inputs (parameters), processes them, and returns an output.

Our isHealthy() function takes three parameters and returns a boolean (true/false) value based on whether all health conditions are met.

function isHealthy(nutrition, oxygen, pH) {
// Check if parameters are in healthy ranges
const goodNutrition = nutrition >= 40;
const goodOxygen = oxygen >= 30;
const goodPH = pH >= 30 && pH <= 70;
// Return true only if ALL conditions are true
return goodNutrition && goodOxygen && goodPH;
}
