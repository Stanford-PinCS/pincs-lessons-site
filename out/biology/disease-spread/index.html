Disease Spread Using BFS

This visualization shows how a disease spreads through a population using the Breadth-First Search (BFS) algorithm.

Each square represents a person. Green squares are healthy, and red squares are infected.

Important: Watch how the disease spreads in complete "waves" day by day - this is exactly how BFS works!

People Infected: 0

How BFS Works in Disease Spread:

  1. The disease starts with one infected person (patient zero) - Day 0.
  2. In the first wave (Day 1), the disease spreads to all direct contacts.
  3. In the second wave (Day 2), it spreads to contacts of those infected on Day 1.
  4. In the third wave (Day 3), it spreads to contacts of those infected on Day 2.
  5. This continues in "waves" or "levels" - exactly how BFS algorithm works!
  6. Each wave corresponds to the distance (shortest path) from patient zero.
  7. The algorithm completes each "level" before moving to the next.

Key BFS Concepts Illustrated:

  • Queue: People waiting to spread the disease
  • Visit Marking: Once infected, you can't be infected again
  • Level-by-level exploration: Complete one wave before starting the next
  • Distance tracking: The wave number shows distance from source

Interactive Controls:
