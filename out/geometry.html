Simple Fractal Generator

Controls how many times the pattern repeats itself

What's Going On Here?

Think of recursion like Russian nesting dolls — it's the same pattern repeated inside itself, getting smaller each time! When you increase the "Recursion Depth" slider, you're telling the computer "repeat this pattern X more times." It's like zooming in on a magical picture that contains copies of itself!

Why This is Amazing:

  • • Look around you! Tree branches, snowflakes, and even clouds follow these patterns
  • • With just a few simple steps, we can create endless complexity
  • • Computer scientists use this "repeat yourself" trick to solve all kinds of problems
  • • The deeper you go, the more fascinating details you discover!
